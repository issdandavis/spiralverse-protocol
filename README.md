## üöÄ Quick Start

```typescript
import { SpiralverseProtocol } from '@spiralverse/core';

const protocol = new SpiralverseProtocol({
  nodeId: 'robot-001',
  tongues: ['command', 'status', 'emergency'],
  vector6D: true
});

// Send a command
await protocol.send({
  tongue: 'command',
  target: 'fleet',
  payload: {
    action: 'navigate',
    coordinates: { x: 100, y: 50, z: 25, t: Date.now(), p: 5, c: 0.95 }
  },
  signatures: [signature1, signature2]
});
```
